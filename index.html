About WINDMI

WINDMI is a low-dimensional nonlinear model of the solar wind–magnetosphere–ionosphere system.

What the tool does

The WINDMI Web Tool reads OMNI solar-wind input over a selected UTC interval, computes the rectified solar-wind driving voltage vBs, solves the eight-state WINDMI ordinary differential equation system, and plots the selected output variables.

WINDMI equations

(1) L dI/dt = vBs(t) − V + M dI1/dt
(2) C dV/dt = I − I1 − Ips − ΣV
(3) 3/2 dp/dt = ΣV2/Ωcps − u0pK∥1/2 θ(I − Ic) − pVAeff/(ΩcpsBtrLy) − 3p/(2τE)
(4) dK∥/dt = IpsV − K∥/τ∥
(5) L1 dI1/dt = V − V1 + M dI/dt
(6) C1 dV1/dt = I1 − I2 − Σ1V1
(7) L2 dI2/dt = V1 − (Rprc + RA2)I2
(8) dWrc/dt = RprcI22 + pVAeff/(BtrLy) − Wrc/τrc

Input and trigger functions

vBs = V0 + Ly|vx||Bz| for Bz < 0; otherwise vBs = V0.
θ(I − Ic) = 1/2 [1 + tanh((I − Ic)/ΔI)].
